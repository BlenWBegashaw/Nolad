# 🏗️ Nolad Business PLC Website

A modern, responsive business website for **Nolad Business PLC**, showcasing construction services, equipment, and company information. Built with clean HTML, CSS, and lightweight JavaScript for fast performance and easy deployment.

---

## 🚀 Live Preview
https://blenwbegashaw.github.io/Nolad/

## 📌 Features

- 🧭 Responsive navigation bar (mobile-friendly)
- 🏗️ Hero section featuring construction machinery (excavators, loaders, trucks)
- 🎯 Modern UI with gradient overlays and clean typography
- 📱 Fully responsive layout (mobile, tablet, desktop)
- 🖼️ Optimized background images for construction theme
- 📄 Multiple sections (About, Services, Contact, etc.)
- ⚡ Lightweight and fast-loading design
